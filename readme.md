# PID Controller App

It is an android app which lets you control **PID** constants through
**Bluetooth** modules like HC-05. [_Download APK_](https://tinyurl.com/PID-Controller)

## Available Commands

| **PARAMETERS** | **INCREMENT** | **DECREMENT** |
|:--------------:|:-------------:|:-------------:|
|     **Kp**     |       P       |       Q       |
|     **Kd**     |       D       |       E       |
|     **Ki**     |       I       |       J       |
|   **Factor**   |       U       |       V       |

**Movement Commands**

- **Start** - S
- **Stop** - T
